# 🚚 - Gerador de Informativos

Sistema web desenvolvido para agilizar a criação de informativos operacionais utilizados em centrais de monitoramento logístico.

O objetivo do projeto é reduzir o tempo gasto na elaboração de textos padronizados, diminuindo erros de digitação e aumentando a produtividade dos operadores.

## ✨ Funcionalidades

- Geração automática de informativos
- Campos dinâmicos conforme o tipo de alerta
- Autocomplete de estados e cidades
- Interface simples e intuitiva
- Botão para copiar o informativo gerado
- Layout responsivo

## 🛠 Tecnologias

- HTML5
- CSS3
- JavaScript

## 🚧 Status do projeto

Em desenvolvimento.

Próximas funcionalidades:

- Histórico de informativos
- Exportação em PDF
- Configuração de modelos personalizados
- Novos tipos de alertas
- Banco de dados
- Login de usuários

## Objetivo

Este projeto foi criado para praticar desenvolvimento Front-end enquanto resolve um problema real do ambiente de trabalho, automatizando uma tarefa repetitiva realizada diariamente.
